# Ecological Growth Models

The repository contains the source code for an app that visualizes growth models in ecology using the example of plankton in a lake. It was developed as part of the SimBioSe project under the “Digital Learning and Teaching” fund at TU Dresden.

# Development Fork!

Please note: This repository contains an unstable developer version. The source code of the stable version can be found at https://gitlab.hrz.tu-chemnitz.de/simbiose/simbiose-w


## Try the app

The [app](https://weblab.hydro.tu-dresden.de/app/simbiose-w/) runs without installation in a web browser on Windows, Linux, iOS, and Android. For the best viewing experience, a sufficiently large screen (laptop, PC, tablet) is recommended.

* Click here for the app: [https://weblab.hydro.tu-dresden.de/app/simbiose-w/](https://weblab.hydro.tu-dresden.de/app/simbiose-w/)

## Implementation

The implementation was carried out using the [**R**](https://www.r-project.org) programming language and the [Shiny](https://shiny.posit.co/) package for interactive web graphics. Typesetting and layout were performed using the [Quarto](https://quarto.org/) publishing system. The [deSolve](https://cran.r-project.org/package=deSolve) package is used for the numerical simulations, and the [ggplot2](https://cran.r-project.org/package=ggplot2) and [plotly](https://CRAN.R-project.org/package=plotly) packages are used for visualization.
The system diagrams were created using the [DiagrammeR](https://cran.r-project.org/package=DiagrammeR) package.
